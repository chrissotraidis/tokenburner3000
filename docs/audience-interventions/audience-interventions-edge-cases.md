# Audience Interventions — Edge Cases

## Action During Final Transition
**Scenario:** User acts while the timer reaches zero.
**Why it matters:** The action could alter a completed score.
**How we handle it:**
1. Fight completion closes intervention input first.
2. Late commands are rejected without charge consumption.
**Status:** 🟢 Implemented

---

## Reduced Effects Tomato
**Scenario:** Throw Tomato is used with effects reduced.
**Why it matters:** Its particle gag is unavailable.
**How we handle it:**
1. Show a static impact notice in the reserved fight rail.
2. Announce the same event text.
3. Apply identical mechanics.
**Status:** 🟢 Implemented
