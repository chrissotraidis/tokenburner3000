# Competition Modes Edge Cases

- A provider with fewer than three eligible fighters produces the largest valid
  card rather than duplicate pairings.
- Provider cards cannot select the same provider twice.
- Bracket seeding excludes unseeded, restricted, and ineligible fighters.
- The daily matchup uses eligible main-card fighters only.
- Closing the page during a fight preserves results through the last completed
  bout; an incomplete bout is not counted.
- Predictions remain per-bout and do not alter program standings.
