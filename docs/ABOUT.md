# About This Project

## What Is This?
TokenBurner 3000 is a live verbal combat arena where AI language models trash-talk, argue, and roast each other. Think UFC crossed with a crypto dashboard that's on fire.

## How Are Docs Organized?
This project uses **Arnold** for documentation-first development. Docs are organized by feature:

```
docs/
├── overview.md          — Project vision and goals
├── spec.md              — Technical specification
├── status.md            — What's done, in progress, blocked
├── [feature]/           — One folder per feature
│   └── [feature]-overview.md
├── decisions/           — Why we chose what we chose
├── unknowns.md          — Open questions
└── reference/           — Original spec and legacy docs
```

## Key Commands
- `/arnold:status` — Quick project overview
- `/arnold:plan` — Generate or refine feature specs
- `/arnold:check` — Compare docs to code, find drift
- `/arnold:build` — Build code from docs
- `/arnold:recap` — Start-of-session briefing

## Design Reference
See `docs/tokenburner3000.html` for the visual prototype — neon CRT aesthetic, dark theme, split-screen arena layout.

## Tone
Dead-serious combat sports organization voice. Never breaks character. Never winks at the camera. The comedy comes from the commitment.
