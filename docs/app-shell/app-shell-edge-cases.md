# App Shell Edge Cases

## Navigation During A Fight

Leaving through the menu cancels the active Exhibition without creating a fight
record. The menu labels this consequence before navigation.

## Narrow Viewport

The command bar keeps the menu, compact brand, current view, and mute control.
Secondary settings remain in the drawer. No range input floats over content.

## Long View Names

The current-view label truncates independently; menu and audio controls remain
fully visible and reachable.

## Reduced Motion

The drawer uses no required animation. Notification changes remain readable as
text and are not accompanied by synthesized sound.
