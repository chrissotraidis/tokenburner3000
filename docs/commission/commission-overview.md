# The Commission

## What It Does

The Commission is a third actor that issues rare regulatory rulings during
Exhibition fights. It satirizes verified availability, gating, fallback, and
regional-access stories without silently invalidating the user's choices.

## Core Rules

- Commission events use the shared fight-event engine. (decided)
- Default frequency is low and disclosed on the Tale of the Tape. (decided)
- Pure Competition rules disable Commission events. (user-stated)
- Every interruption has a recovery, tag-in, or compensation path. (decided)
- Region Lock requires an explicit jurisdiction modifier. (user-stated)
- Restricted/no-show fighters cannot block ordinary setup. (decided)
- Factual lore uses verified source metadata; mechanics are marked satirical. (decided)

## Initial Rulings

- Export Control — Fable temporarily tags to Opus.
- Government Hold — GPT-5.6 Sol shifts to Terra or Luna.
- Region Lock — only possible in a disclosed EU venue modifier.
- Delayed — reserved for Gemini 3.5 Pro special presentation.
- Capability Review — temporarily suppresses a signature, never the whole UI.

## Acceptance Criteria

- [x] Commission events can be enabled or disabled before a fight.
- [x] Risk is disclosed before confirmation.
- [x] No ruling leaves a fighter stream blank.
- [x] Every ruling appears in commentary, live presentation, and the final timeline.
- [x] Region Lock cannot occur without an EU venue modifier.
- [x] Pure Competition produces no Commission events.
